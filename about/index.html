Work Experience

I currently work developing software for a tier 1 automotive supplier. In my time working for this company, I have learned how to interface with different vehicle CAN bus architectures, solve timing related issues, and write C++ code which is optimal for running on an embedded system.

Education

I graduated from Lawrence Technological University in 2014 with a BS in Computer Science. My concentrations in school were business software development and embedded systems.

Projects

I work on a number of hobby projects in my spare time. These projects range from programming AVR microcontrollers to building custom cross-compilers and Linux systems. I plan on making this blog a space where I can document lessons learned while working on my own projects.
